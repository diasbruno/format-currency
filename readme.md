# format-currency

a basic currency formatter.

## api

- value : the value to format.
- decimalChar : punctuation for decimal.
- fractionChar : punctuation for fration places.
- decimalPlaces : number of decimal places.

```
formatCurrency(
  value : String | Number,
  decimalChar : Char,
  fractionChar : Char,
  decimalPlaces : Number
) : String
```
